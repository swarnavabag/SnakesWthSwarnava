# Snakes With Swarnava

A classic **Snake Game** built using **Python** and **Pygame**, created as part of my early game-development journey.

This project recreates the timeless Snake gameplay where the player controls a growing snake, collects food, increases score, and avoids collisions with walls or itself.

---

## Features

* Smooth keyboard-controlled snake movement
* Real-time score tracking
* Persistent high-score saving
* Sound effects for better gameplay feedback
* Custom assets including fonts and images

---

## Tech Stack

* **Python**
* **Pygame**
* File Handling (for score persistence)

---

## Concepts Practiced

Through this project, I explored and implemented:

* Game loop architecture
* Event-driven programming
* Collision detection logic
* State management
* Basic asset handling in games
